Gesture based navigation system

"How can we enable seamless, non-contact interaction in specialized environments to overcome the physical limitations of traditional input devices?"

Problem Statement

Modern digital environments increasingly require non-contact interaction (e.g., dynamic presentations, sterile medical settings, or touchless kiosks). Traditional inputs like mice and keyboards are physically limiting, unhygienic in shared spaces, and inefficient in contexts where spatial freedom is paramount.

Literature Review / Market Research

O. M. Vultur, Ş. G. Pentiuc and A. Ciupu, "Navigation system in a virtual environment by gestures," 2012 9th International Conference on Communications (COMM), Bucharest, Romania, 2012, pp. 111-114.

Research Gap / Innovation

Bridging the gap between high-latency sensor-based systems (like Kinect) and real-time, lightweight camera-driven gesture recognition using YOLOV11.

System Methodology

Dataset / Input

Dataset obtained from RoboFlow and annotated manually for gesture precision.

Model / Architecture

Custom YOLOV11 Machine Learning model running in tandem with MediaPipe and OpenCV.

Live Execution

Results & Analysis

Accuracy / Performance 94.2%

"The system achieves high-precision gesture mapping with minimal latency, significantly outperforming standard mouse-emulation software in specialized spatial tasks."
